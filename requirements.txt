criteria 1
    in Triage phase:
        if the cbd is unknown, then the owner will be set to "CIHT". Should be re-evaluated once cbd is set to a value
        else if incident.properties.cbd == "GWM US" then set owner to "DISO-GWM US" for rest of lifecycle
        else if incident.properties.impact_rating == 0 then set owner to "DISO-CIHT"
        else if incident.properties.impact_rating == 1 or 2 then set owner to "DISO-CIHT"
        else if incident.properties.impact_rating >= 3 then set owner to respective DISO (f"DISO-{incident.properties.cbd}")

criteria 2
    after Triage phase, including Response and Recovery and the rest of the incident lifecycle:
        if incident.properties.cbd is any of ["AM", "P&C", "GWM", "GWM WMI"] and incident.properties.impact_rating is 1 or 2 then set owner to respective business DISO and members = "CIHT" + current members
        if incident.properties.cbd is any of ["GF", "IB"] and incident.properties.impact_rating is 1 or 2 then set owner to "DISO-CIHT" and members unchanged
        if incident.properties.impact_rating >= 3 then set owner to respective DISO (f"DISO-{incident.properties.cbd}") and members = "CIHT" + current members
        if incident.properties.impact_rating == 0 then set owner to "DISO-CIHT" except when incident.properties.cbd == "GWM US"

criteria 3
    once phase moves out of Triage, ownership follows criteria 2 for the rest of the incident lifecycle
    once Response and Recovery is complete, there is no separate ownership reset; the rest-of-lifecycle ownership remains as-is
    once task is closed, there is no separate ownership reset; the rest-of-lifecycle ownership remains as-is

criteria 4
    after Triage phase, owner should stay with CIHT in the below conditions unless impact rating is set to 3 or above or cbd is changed to anything other than "GF"
    if incident.properties.causedby = "IT systems" or "Other third party" then owner set to "CIHT"
    if incident.properties.causedby = "Employee" or "Other third party" and incident.properties.type = "cyber attack" or "cyber incident" then owner set to "CIHT"

criteria 5
    if need to transfer case from "X" to "Y", the logic set for the new owner should apply for the rest of the case lifecycle, regardless of the original owner and the new owner

criteria 6
    if the impact rating is 0, the owner will always be "CIHT" except in case of incident.properties.cbd == "GWM US". This applies regardless of phase
